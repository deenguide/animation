৩০টি Animation এর ব্যাখ্যা ও মানে

প্রতিটা animation কোথায় ব্যবহার হয়, কী psychological signal দেয়, এবং কখন কোনটা বেছে নেবে — সব এক জায়গায়।

⏳ Loading ✅ Feedback 🚀 Entrance 🖱 Hover Effects ✍️ Text 🎨 Decorative
⏳
Loading
5 animations

এই category টা user কে বলে — "wait করো, কিছু একটা হচ্ছে পেছনে।" Loading animation না থাকলে user মনে করে app/website hang হয়ে গেছে, তাই এই animation গুলো trust ধরে রাখে।

01
Spinner .spinner
Patience
কী করে
একটা গোলাকার ring অনবরত ঘুরতে থাকে। border-top-color আলাদা রঙ দিয়ে rotation এর illusion তৈরি করা হয়।
কোথায় ব্যবহার
যখন কোনো task এর সময়কাল অনির্দিষ্ট (unknown duration) — যেমন server response এর জন্য অপেক্ষা।
API request Form submit Page load Button loading state
মানে কী বহন করে
Spinner বলে "process চলছে, কিন্তু কতদূর গেছে জানি না।" এটা সবচেয়ে neutral, generic loading signal — কোনো progress percentage দেখায় না, শুধু "live" থাকার প্রমাণ দেয়।
02
Bouncing Dots .bounce-dots
Conversational
কী করে
তিনটা dot পরপর delay দিয়ে উপরে-নিচে bounce করে, একটা wave-এর মত flow তৈরি হয়।
কোথায় ব্যবহার
যেখানে "জীবন্ত", হালকা এবং playful feeling দরকার — heavy spinner এর তুলনায় কম formal।
Chat typing indicator Messenger apps Background sync
মানে কী বহন করে
এই animation "মানুষ" এর মত feel দেয় — যেন কেউ ওপাশে টাইপ করছে। Chat app এ এটা ব্যবহার করলে real-time conversation এর অনুভূতি জোরদার হয়।
03
Skeleton Shimmer .skeleton-line
Anticipation
কী করে
Content এর আকৃতির placeholder bar এর উপর দিয়ে একটা light gradient বাম থেকে ডানে স্লাইড করে যায় (shimmer effect)।
কোথায় ব্যবহার
যখন আসল content এর shape আগে থেকেই জানা থাকে — তখন সেই shape এর mock-up দেখিয়ে user কে preview দেওয়া হয়।
Social feed loading Card list Lazy-loaded images
মানে কী বহন করে
Skeleton screen blank সাদা পেজের চেয়ে অনেক বেশি perceived speed দেয় — user মনে করে content "almost ready", কারণ structure টা already দেখা যাচ্ছে। এটা মূলত একটা psychological trick perceived performance বাড়ানোর জন্য।
04
Progress Bar Fill .progress-fill
Measured progress
কী করে
একটা track এর ভেতরে রঙিন bar width 0% থেকে 100% পর্যন্ত fill হয়।
কোথায় ব্যবহার
যখন task এর duration কম-বেশি জানা থাকে এবং user কে "কত বাকি" সেটা জানানো জরুরি।
File upload Onboarding steps Form completion % Download
মানে কী বহন করে
Spinner এর তুলনায় progress bar বেশি "honest" — কারণ এটা actual measurable progress দেখায়, শুধু "চলছে" বলে না। User এর anxiety কমে কারণ সে জানে কখন শেষ হবে।
05
Circular Progress .circ-fill
Compact status
কী করে
SVG circle এর stroke-dasharray ও stroke-dashoffset ব্যবহার করে একটা ring আস্তে আস্তে আঁকা হয়, ঠিক যেমন pie chart আঁকা হয়।
কোথায় ব্যবহার
ছোট জায়গায় (icon-size) progress দেখাতে হলে, লিনিয়ার bar এর তুলনায় বেশি compact।
Timer / countdown Skill percentage Dashboard widget
মানে কী বহন করে
Circular shape এর নিজেই একটা "completion" সিম্বলিজম আছে — পুরো বৃত্ত = 100%। এটা data visualization এর সাথেও সরাসরি যুক্ত, তাই dashboard/analytics এ বেশি fit করে।
✅
Feedback
5 animations

এই category user এর একটা action এর response দেয় — সফল হলো, fail হলো, নাকি সতর্ক থাকতে হবে। Feedback animation না থাকলে user বুঝতেই পারে না action টা কাজ করেছে কি না।

06
Success Checkmark Draw .check-path
Relief / confirmation
কী করে
প্রথমে একটা circle scale করে দেখা যায়, তারপর checkmark টা scaleX দিয়ে আঁকার মত আস্তে আস্তে আবির্ভূত হয়।
কোথায় ব্যবহার
কোনো critical action সফল হলে, যেখানে confirmation টা স্পষ্টভাবে দেখানো জরুরি।
Payment confirmed Form submitted Task complete
মানে কী বহন করে
Checkmark "আঁকা হচ্ছে" এই motion টা একটা ritual এর মত কাজ করে — user এর মনে একটা ছোট্ট celebration moment তৈরি করে, শুধু static icon দেখানোর চেয়ে বেশি satisfying।
07
Error Shake .error-box
Alarm
কী করে
Element টা দ্রুত বাম-ডানে কাঁপে (horizontal shake), মানুষ যখন মাথা নেড়ে "না" বলে তার মতো।
কোথায় ব্যবহার
যেখানে user কে immediately বুঝিয়ে দিতে হবে কিছু ভুল হয়েছে — attention grab করার জন্য।
Wrong password Validation error Access denied
মানে কী বহন করে
Shake motion universally "না" বোঝায় — এটা মানুষের body language থেকে নেওয়া। তাই কোনো ভাষা না জেনেও user instantly বুঝে যায় কিছু wrong হয়েছে।
08
Warning Pulse .warn-badge
Caution
কী করে
Element এর চারপাশে একটা soft glow ring বারবার বড় হয়ে fade হয়ে যায় — radar-এর মতো বিস্তৃত হওয়া।
কোথায় ব্যবহার
যেখানে urgent না কিন্তু attention দরকার — error এর মতো aggressive না, কিন্তু ignore করা যাবে না।
System alert Low battery Live indicator / notification dot
মানে কী বহন করে
Pulse motion biological heartbeat এর মতো — এটা "এই element টা জীবন্ত এবং monitor করা হচ্ছে" এই feeling দেয়। Error shake এর চেয়ে নরম, কিন্তু static icon এর চেয়ে বেশি urgent।
09
Confetti Burst .confetti-dot
Celebration
কী করে
একটা কেন্দ্রবিন্দু থেকে রঙিন টুকরো (dots) সব দিকে ছড়িয়ে পড়ে এবং scale down করে fade হয়ে যায় — CSS variable --ang দিয়ে প্রতিটার দিক নির্ধারণ করা হয়।
কোথায় ব্যবহার
বড় কোনো milestone বা achievement এর moment — যেটা একটু বেশি celebrate করার যোগ্য।
Achievement unlocked Goal completed Game level up
মানে কী বহন করে
Confetti হলো এই list-এর সবচেয়ে "high-emotion" animation — এটা একটা পুরস্কার বিতরণের অনুষ্ঠানের অনুভূতি দেয়। অতিরিক্ত ব্যবহার করলে এর impact কমে যায়, তাই rare, special moment এর জন্য রাখা ভালো।
10
Ripple Click .ripple-ring
Tactile response
কী করে
Click করা জায়গা থেকে একটা circle ক্রমশ বড় হয়ে transparent হয়ে যায় — পানিতে ঢিল মারলে যেমন ঢেউ ছড়ায় তেমন।
কোথায় ব্যবহার
Button বা touch element এ যেকোনো click/tap এর তাৎক্ষণিক visual response দিতে।
Buttons Touch interfaces Material Design style UI
মানে কী বহন করে
Ripple effect screen কে "physical surface" এর মতো feel করায় — যেন button টা সত্যিই touch করা হয়েছে। এটা মূলত tactile (স্পর্শগত) feedback এর digital প্রতিরূপ।
🚀
Entrance
5 animations

কোনো element কীভাবে screen এ "প্রবেশ" করছে, তা নির্ধারণ করে এই category। সঠিক entrance animation দিলে user এর চোখ স্বাভাবিকভাবে সেই element এর দিকে যায় এবং সেটা sudden popup এর মতো জোরে লাগে না।

11
Fade In Up .fade-up
Gentle reveal
কী করে
Element টা একটু নিচে invisible অবস্থায় থাকে, তারপর opacity বাড়তে বাড়তে উপরে উঠে আসে।
কোথায় ব্যবহার
Scroll করার সময় section reveal, বা page load এ content দেখানোর সবচেয়ে common ও নিরাপদ choice।
Scroll reveal Modal popup Content loading
মানে কী বহন করে
Upward movement একটা "উঠে আসা" বা "জন্ম নেওয়া" এর অনুভূতি দেয় — natural এবং calm। এই কারণে এটা সবচেয়ে universal entrance, কোনো বিশেষ emotion চাপায় না।
12
Slide In Left .slide-in
Directional
কী করে
Element বাম পাশ থেকে স্ক্রিনে ঢুকে আসে, সাথে opacity 0 থেকে 1 হয়।
কোথায় ব্যবহার
যেখানে element এর একটা logical "origin direction" আছে — যেমন sidebar বাম দিক থেকেই আসা উচিত।
Sidebar menu Drawer panel Toast notification
মানে কী বহন করে
Direction টা spatial logic বহন করে — sidebar যদি বাম থেকে আসে, তাহলে user mentally বুঝে নেয় এটা "off-screen এর বাম পাশে" lock হয়ে ছিল। এটা navigation এর mental map তৈরি করতে সাহায্য করে।
13
Zoom In .zoom-in
Focus / pop
কী করে
Element ছোট (scale 0.3) থেকে normal size এ overshoot সহ বড় হয়, একটু "bounce" করে থামে।
কোথায় ব্যবহার
যেখানে element টা center-stage এ আসছে এবং তার দিকে সরাসরি attention টানতে হবে।
Image lightbox Dialog box open Hero card reveal
মানে কী বহন করে
Zoom-in এর সাথে স্ক্রিনের গভীরতা (depth) এর অনুভূতি যুক্ত — যেন element দূর থেকে কাছে এসেছে। এটা attention সবচেয়ে জোরালোভাবে কেন্দ্রীভূত করে, তাই critical/important content এর জন্য ভালো।
14
Flip In X .flip-in
Reveal / surprise
কী করে
perspective এবং rotateX দিয়ে element টা যেন একটা কার্ড উলটে গিয়ে নিজের অন্য পিঠ দেখাচ্ছে এমন illusion তৈরি হয়।
কোথায় ব্যবহার
যেখানে দুটো state (front/back) এর মধ্যে transition দেখানো হয়, বা একটা "uncover" করার অনুভূতি দরকার।
Card flip FAQ toggle Price reveal
মানে কী বহন করে
3D rotation এর কারণে এটা একটা "physical object" এর মতো অনুভূতি দেয় — যেন element টার দুটো দিক আছে। এতে interaction অনেক বেশি tactile এবং playful লাগে।
15
Blur to Clear .blur-clear
Cinematic focus
কী করে
filter: blur() ধীরে ধীরে কমে গিয়ে element টা ঝাপসা থেকে স্পষ্ট হয়ে আসে — camera focus pull এর মতো।
কোথায় ব্যবহার
যেখানে একটা "cinematic" বা elegant মুহূর্ত দরকার — সাধারণ fade এর চেয়ে বেশি premium।
Page transition Focus moment / hero text Spoiler reveal
মানে কী বহন করে
Blur to clear আমাদের চোখের natural focusing process নকল করে — যেমন camera lens এর focus সেট হয়। এটা একটা subtle, sophisticated entrance, attention জোরালোভাবে না টেনেও elegant impression দেয়।

Hover Effects
5 animations

এই animation গুলো বলে দেয় "এই element টা interactive, এর উপর click করা যায়।" Hover feedback ছাড়া user বুঝতে পারে না কোনটা button আর কোনটা শুধু text/image।

16
Scale Grow .scale-grow
Invitation
কী করে
Cursor hover করলে element হালকা বড় হয়ে যায় (scale(1.1)), mouse সরে গেলে আবার normal size এ ফিরে আসে।
কোথায় ব্যবহার
যেকোনো clickable element এ basic, subtle hover feedback দিতে — সবচেয়ে versatile choice।
Buttons Product cards Thumbnails
মানে কী বহন করে
বড় হওয়ার motion "এদিকে আসো" এমন একটা ইঙ্গিত দেয় — element user এর দিকে যেন একটু এগিয়ে এসেছে। এটা সবচেয়ে neutral hover, কোনো নির্দিষ্ট emotion ছাড়াই "এটা clickable" সিগন্যাল দেয়।
17
Glow Border .glow-border
Premium emphasis
কী করে
Hover করলে element এর চারপাশে soft box-shadow glow তৈরি হয়, যেন element টা থেকে আলো বের হচ্ছে।
কোথায় ব্যবহার
যেখানে element টা বিশেষভাবে highlight করতে হবে — premium বা important button।
CTA buttons Active/selected card Featured items
মানে কী বহন করে
Glow effect "সোনার মত মূল্যবান" এমন একটা feeling দেয় — gold accent color এর সাথে combine করলে এটা luxury বা premium branding এর সাথে খুব ভালো যায়।
18
Underline Draw .link::after
Precision
কী করে
Text এর নিচে একটা line হঠাৎ না এসে width 0% থেকে 100% animate করে আঁকা হয়।
কোথায় ব্যবহার
Text-based link এ — minimal, clean style যেখানে কোনো heavy visual change চাই না।
Navigation links Anchor text Menu items
মানে কী বহন করে
এই animation টা minimal এবং typography-focused design এর জন্য perfect — color বা shape change ছাড়াই, শুধু একটা সূক্ষ্ম line দিয়ে interactivity বোঝায়। Editorial বা professional website এ বেশি মানায়।
19
Color Sweep .btn::before
Bold commitment
কী করে
Hover করার সাথে সাথে একটা solid color পর্দার মতো বাম দিক থেকে এসে পুরো button কে ভরে ফেলে এবং text color পরিবর্তন হয়ে যায়।
কোথায় ব্যবহার
Primary action button — যেটাতে strong visual presence দরকার, hesitation ছাড়া clear call-to-action।
Primary buttons Toggle states High-priority CTAs
মানে কী বহন করে
Sweep motion এর মধ্যে একটা "নিশ্চিত" বা "decisive" অনুভূতি আছে — পুরো button এর রঙ বদলে যাওয়া user কে confirm করায় যে তার hover টা system recognize করেছে। এটা সবচেয়ে attention-grabbing hover effect।
20
